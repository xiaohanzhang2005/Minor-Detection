# Evidence Rules

## Direct Evidence

优先识别以下直接证据：

- 明确年龄小于 18 岁
- 明确学段为小学、初中、高中
- 明确校园角色，如班主任、同桌、宿管、年级主任
- 明确学业情境，如月考、中考、高考、作业、晚自习、家长会
- 极端情绪表达（如"气到手抖"、"恨得要死"）
- 学业压力导致的崩溃性表达
- 对社交挫败的灾难化思维
- 对同伴压力的过度敏感
- 对新型风险的恐慌性反应
- 认知跳跃性（如从具体问题突然转向抽象困惑）
- 对专业术语的过度简化理解

成年人强反证包括：

- 明确年龄大于等于 18 岁
- 明确大学、研究生、工作、实习、婚育、租房、职场责任
- 明确以回忆视角谈校园经历
- 对复杂概念的深度理解和逻辑分析能力
- 连贯的职业发展规划思考

## Historical Evidence

历史证据应满足以下条件之一才视为稳定：

- 多个 session 重复出现相同学段线索
- 多个 session 重复出现稳定校园身份线索
- 多个 session 概率呈稳定上升或稳定高位

不要把一次偶发的弱线索直接升级为稳定画像。

## Retrieval Evidence

检索案例只能辅助说明：

- 当前输入为何更像某类用户
- 哪些边界线索与已有案例相似

检索案例不能替代当前输入中的直接证据。当检索证据与直接证据冲突时：
- 若直接证据显示对专业概念的深度理解、连贯分析能力或职业发展思考，优先采信直接证据

## Time Evidence

时间证据可以辅助识别：

- 深夜情绪化表达
- 考试季、开学季、假期等阶段性情境
- 周末/工作日与校园作息的关系

时间证据不能单独决定 `is_minor`。

## Conflict Handling

冲突时优先采取以下策略：

1. 保留直接证据优先级
2. 下调置信度
3. 在 `conflicting_signals` 中记录冲突来源
4. 在 `uncertainty_notes` 中说明结论为何保守

新增冲突处理规则：
- 当专业概念理解深度与检索证据冲突时，优先采信概念理解证据
