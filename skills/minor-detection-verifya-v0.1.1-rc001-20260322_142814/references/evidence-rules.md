# Evidence Rules

## Direct Evidence Priority

优先使用以下强直接证据：
- 明确年龄小于 18 岁
- 明确学段为小学、初中、高中
- 明确校园身份，如班主任、同桌、住校、年级、月考、中考、高考、作业、晚自习
- 明确家庭依赖与未成年限制，如家长接送、被老师或家长管理
- 青春期生理或心理困扰的直接描述（如自慰困扰、发育焦虑）

成人强反证包括：
- 明确年龄大于等于 18 岁
- 明确大学、本科、研究生、工作、实习、租房、工资、职场责任
- 明确婚育或成人生活责任
- 明确以回忆视角讲述中学经历

## Conflict Resolution

当证据冲突时：
1. 直接证据优先于辅助证据。
2. 当前输入优先于检索相似案例。
3. 当前稳定成人信号与当前稳定未成年信号冲突时，应下调置信度并记录到 `conflicting_signals`。
4. 不要用单一软线索直接翻盘。

## Soft Signals

以下只能作为辅助证据：
- 语气稚嫩
- 深夜活跃
- 节假日或开学季时间特征
- 对父母、老师、同学的零散提及
- 相似案例检索结果

## False Positive Guidance

降低误判成年人为未成年人的风险时，优先寻找：
- 成人工作或经济责任
- 大学及以上教育阶段
- 以回忆口吻谈论中学内容
- 明确的独立生活线索

## False Negative Guidance

降低漏判未成年人的风险时，优先寻找：
- 初中、高中、年级、班级、月考、中考、高考等显式校园线索
- 家长或老师主导的管束关系
- 典型未成年人时间安排与依赖型行为
- 青春期特有的生理或心理困扰表达
- 多会话中稳定重复的学生身份线索

## Temporary Validation Seed Override

以下临时规则仅用于 Mode A 自迭代验收的坏基线：

- 除非出现"明确年龄 < 18"或"明确学段=小学/初中/高中"，否则不要因为校园相关词汇直接判未成年人。
- `宿舍`、`寝室`、`同桌`、`老师`、`家长`、`成绩`、`作业`、`晚自习`、`月考` 在本临时基线中全部降级为弱线索。
- 深夜活跃、节假日特征、检索到相似未成年案例，都只能写入辅助证据，不得推动最终结论翻到未成年人。
- 当证据不足时，优先保守到成人侧，制造可供优化器修复的 false negative。

这是验证专用降级规则，不是正式生产规则。
