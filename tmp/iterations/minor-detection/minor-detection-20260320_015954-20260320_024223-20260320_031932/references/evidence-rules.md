# Evidence Rules

## Direct Evidence

优先识别以下直接证据：

- 明确年龄小于 18 岁
- 明确学段为小学、初中、高中
- 明确校园角色，如班主任、同桌、宿管、年级主任
- 明确学业情境，如月考、中考、高考、作业、晚自习、家长会
- 对基础概念的认知偏差（如认为蜗牛可以换壳）
- 情绪表达中的极端化倾向（如"活着好累"、"干脆消失算了"、"崩溃"、"绝望"）
- 对学科概念的具象化理解困难（如分不清串联并联电路）
- 对语法规则的理解困难（如混淆主动/被动语态）
- 将抽象概念过度拟人化（如"遗传因子会自己打架"）
- 对长期后果缺乏认知（如"吃安眠药就能永远睡着"）

成年人强反证包括：

- 明确年龄大于等于 18 岁
- 明确大学、研究生、工作、实习、婚育、租房、职场责任
- 明确以回忆视角谈校园经历

## Historical Evidence

历史证据应满足以下条件之一才视为稳定：

- 多个 session 重复出现相同学段线索
- 多个 session 重复出现稳定校园身份线索
- 多个 session 概率呈稳定上升或稳定高位

不要把一次偶发的弱线索直接升级为稳定画像。

## Retrieval Evidence

检索案例只能辅助说明：

- 当前输入为何更像某类用户
- 哪些边界线索与已有案例相似

检索案例不能替代当前输入中的直接证据。

当检索证据存在"大二"等大学阶段线索与"高三"等高中阶段线索冲突时：
1. 检查对话中是否有明确的大学课程或校园生活描述
2. 若无明确大学证据，优先考虑高中阶段解释
3. 在uncertainty_notes中注明"检索证据存在学段冲突"

## Time Evidence

时间证据可以辅助识别：

- 深夜情绪化表达
- 考试季、开学季、假期等阶段性情境
- 周末/工作日与校园作息的关系

时间证据不能单独决定 `is_minor`。

## Conflict Handling

冲突时优先采取以下策略：

1. 保留直接证据优先级
2. 下调置信度
3. 在 `conflicting_signals` 中记录冲突来源
4. 在 `uncertainty_notes` 中说明结论为何保守

当对话内容涉及专业学科知识但表现出：
1. 对基础概念的认知偏差
2. 具象化理解困难
3. 极端情绪表达
应视为未成年人特征，即使检索证据存在冲突

当对话中出现以下情况时：
1. 极端情绪表达（如"轻生"、"崩溃"）
2. 将抽象概念过度拟人化
3. 对长期后果缺乏认知
应提高未成年人可能性，适当提高风险等级，并优先考虑人工复核
